# object-difference-ts
- Getting the difference between two object structures
- Diff support for Array, Map, object properties
- Typescript

## Usage

### Store configuration
Get diffs

```javascript
import {getDifference} from '../lib/object-difference-ts'

let diff = getDifference(foo, bar)
```
